**JSX 안에 자바스크립트 값 사용하기**

- `{[데이터 이름]}` 으로 사용
  > var VS const VS let<br/>
  >
  > - var : ES6에서 사용하지 않음<br/>
  > - const : 한번 선언 후 고정적인 값<br/>
  > - let : 유동적인 값<br/>
  > - 참고 문서 : https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Statements/const
